English | 简体中文

一个基于 Vue 3 和 Tailwind CSS 构建的高保真微信模拟器（单文件 HTML 版）。
主打 “精装朋友圈” 概念，内置强大的 “上帝模式” (God Mode) 控制台，让你随心所欲修改余额、伪造聊天记录、定制朋友圈互动。

✨ 核心特性 (Features)

🎨 像素级还原 (Pixel-Perfect UI)

深度复刻微信 8.0+ 界面风格（WeUI）。

包含微信、通讯录、发现（朋友圈/视频号/小程序）、我（钱包/服务）四大核心模块。

双端适配：支持一键切换 手机端 (Mobile) 和 PC 端 视图。

👑 上帝模式 (God Mode)

可视化控制台：点击右下角皇冠图标呼出。

万物皆可改：实时修改钱包余额、微信号、昵称、手机电量、系统时间。

聊天操控：伪造对话（自选发送者是“我”还是“对方”）、发送文本/图片/表情/语音。

朋友圈操控：伪造朋友圈内容、时间、点赞人名、评论内容（支持伪造任意好友评论）。

🛠 技术栈 (Tech Stack)

Vue 3 (Composition API)：响应式数据驱动。

Tailwind CSS：原子化 CSS，快速构建 UI。

Single File HTML：零依赖、零后端。所有代码（HTML/JS/CSS）均在一个 .html 文件中，双击即可运行。
