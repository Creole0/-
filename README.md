## 📖 项目简介 (Introduction)

Demo: https://44512c6d.htmlautodeploy.pages.dev

这不仅仅是一个 UI 界面，这是一个 **“精装朋友圈”生成器**。

基于 **Vue 3** 和 **Tailwind CSS** 构建，采用单文件 HTML 架构（**零依赖，下载即用**）。它完美复刻了微信 8.0+ 的界面风格，并内置了强大的 **“上帝模式” (God Mode)**。你可以把它当做前端学习案例，也可以用来生成各种“梦想中”的聊天记录和余额截图。

## ✨ 核心特性 (Features)

### 🎨 像素级还原 (Pixel-Perfect UI)
* 深度复刻微信 8.0+ 界面风格（WeUI）。
* **四大模块齐全**：微信、通讯录、发现（朋友圈/视频号/小程序）、我（钱包/服务）。
* **双端适配**：内置 PC 端和移动端两套布局，一键切换。

### 👑 上帝模式 (God Mode)
点击屏幕右下角的 👑 皇冠图标，即可打开上帝控制台：

* **💰 财富自由**：钱包余额想填多少填多少。
* **🎭 身份伪造**：随意修改昵称、微信号、头像。
* **🗣️ 聊天操控**：在聊天框中，你可以自由切换 **“我”或“对方”** 的身份发送消息。
* **📸 朋友圈导演**：
    * 伪造任意人的朋友圈内容。
    * **操控互动**：给任意朋友圈添加任意名字的点赞和评论（例如让“马化腾”给你点赞）。

## 🛠 技术亮点 (Tech Stack)

* **Vue 3 (Composition API)**：清晰的响应式数据流。
* **Tailwind CSS**：原子化 CSS，极速构建复杂 UI。
* **Single File HTML**：黑科技打包。HTML、CSS、JS 全部内联在一个文件中，**无需 npm install，无需服务器，双击 HTML 文件即可运行。**

## 🚀 快速开始 (Quick Start)

### 方式一：小白极速版 (推荐)
1.  **下载**：直接下载本仓库中的 `WeChat_Simulator_V9.html` 文件。
2.  **运行**：双击文件，使用 Chrome、Edge 或 Safari 浏览器打开。
3.  **开始**：点击右下角 👑 图标，开始你的表演。

### 方式二：给朋友在线演示 (GitHub Pages)
想发个链接给朋友玩？

1.  将 `WeChat_Simulator_V9.html` 重命名为 `index.html`。
2.  创建一个新仓库，把这个文件传上去。
3.  在仓库 `Settings` -> `Pages` 中开启 GitHub Pages。
4.  发送链接给朋友即可，手机电脑都能访问。

## 🎮 玩法指南 (How to Play)

* **修改余额**：
    * 进入「我」->「服务」，点击绿色的钱包卡片。
    * 或者点击右下角 👑 -> 在控制台输入金额。
* **伪造聊天**：
    * 进入任意聊天框（或点击右上角 `+` 号新建对话）。
    * 打开上帝模式开关。
    * 输入框上方会出现「发送者：我 | 对方」切换按钮。
* **精装朋友圈**：
    * 「发现」->「朋友圈」-> 点击右上角相机。
    * (上帝模式下) 可以选择「发布身份」（伪造老板/前任发朋友圈）。
    * 发布后，点击右下角 `..` 按钮 -> 此时底部会出现 **[上帝操控]** 面板，可随意添加点赞和评论。
